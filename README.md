# RefBoard

<details open>
<summary><b>English / Deployment Guide</b></summary>

### Overview
**RefBoard** is a minimalist, professional reference management tool designed for creators. Inspired by BeeRef, it provides a clean workspace to organize your project references and deadlines.

### Key Features (v1.1.0)
- **Minimalist Dashboard**: Focused on your active projects with a premium aesthetic.
- **Auto-Update**: Stay current with new features and fixes automatically.
- **Reference Board**: Independent infinite canvas for spatial image organization.
- **8-Handle Resizing**: Scale items from any direction while maintaining the perfect aspect ratio.
- **Discord Integration**: Get automated reminders for upcoming deadlines with multilingual notification support.
- **Bilingual Support**: Full support for English and Japanese.
- **Auto-Save**: Securely saves your project data and settings.
- **Global Invoice Generation**: Create professional PDF invoices with a dedicated settings window. Supports multi-currency, custom tax labels/rates, and Qualified Invoice System (Japan).
- **Hybrid File/Folder Handling**: Attach individual files or entire asset folders. Quick-open parent directories of linked files.
- **Single Instance Prevention**: Focuses existing window instead of launching duplicates.

### How to Use
1. **Onboarding**: Select your preferred language and theme on the first launch.
2. **Add Project**: Enter a title and press Enter.
3. **Reference Board**: Click the board icon (ğŸ–¼ï¸) to open the infinity canvas. Drag and drop images (from local or web) to the canvas.
4. **Link Files & Folders**: Attach your work files (.psd, .lip, .mov) or asset folders to quickly open them from RefBoard.
5. **Generate Invoice**: Set up your business and tax info once, then generate PDF invoices with one click.

### Installation
Latest releases are available on [GitHub Releases](https://github.com/rieszedit/refboard/releases). Download the `RefBoard_x.x.x_x64-setup.exe` and follow the installer.
